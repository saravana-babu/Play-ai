# 🚀 Play AI Web Assistant - Installation Guide

Welcome to the **Play AI Web Assistant**. Since we are in the early access phase, the extension is currently available via the GitHub repository. Follow these steps to install it on your Chrome browser.

## 🛠️ Installation Steps

1. **Download the Extension Assets**:
   - Stay in this repository and navigate to the `chrome-extension/public` folder.
   - Alternatively, [Download the repository](https://github.com/saravana-babu/Play-ai/archive/refs/heads/main.zip) and unzip it.

2. **Open Chrome Extensions Page**:
   - Open your Chrome browser and type `chrome://extensions/` in the address bar.
   - Or, click the **Three Dots** (top right) > **Extensions** > **Manage Extensions**.

3. **Enable Developer Mode**:
   - In the top-right corner of the Extensions page, toggle the **Developer mode** switch to **ON**.

4. **Load the Extension**:
   - Click the **"Load unpacked"** button that appears in the top-left.
   - Select the `chrome-extension/public` folder from your downloaded files.

5. **Pin and Setup**:
   - Click the puzzle icon (Extensions menu) in your Chrome toolbar.
   - Find **Play AI Web Assistant** and click the **Pin** icon.
   - Click the extension icon, enter your **API Key** (Groq, Gemini, etc.), and start using the 3x3 Power Grid!

## ✨ Features
- **3x3 High-Density Grid**: 9 specialized tools in one compact side panel.
- **Curate Mode**: Summarize, Extract Insights, Analyze Tone, and "Explain for Kids".
- **Create Mode**: Rewrite Text, Extract Action Items, Generate Tweets/Emails, and Interview Prep.
- **Groq Support**: Experience ultra-fast inference with Llama 3.3 and Mixtral integration.
- **Obsidian UI**: Sleek 280px wide glassmorphism design with a pinned footer.
- **Keyboard Shortcut**: Use `Alt + Shift + Space` to toggle the assistant instantly.

## 🔒 Security
- **BYOK (Bring Your Own Key)**: Your API keys are stored **locally** in your browser's secure storage.
- **Privacy**: We never send your data or keys to any external server except directly to the AI provider you select.
